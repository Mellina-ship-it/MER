# MER
modelo relacional (MER)
Modelo Relacional construido do zero a pedido na máteria de Banco de Dados
Descrição: 
Tema: Sistema de gerenciamento de jogos digitais da empresa EA
1. Contextualização:
A Eletronic Arts (EA) é uma empresa do setor de games digitais que atua no desenvolvimento, publicação e comercialização de jogos para diferentes plataformas e públicos. Para que essa infraestrutura funcione, é necessário o controle de informações sobre franquias, jogos lançados, estúdios responsáveis pelo desenvolvimento, funcionários envolvidos nos projetos, plataformas em que os jogos são disponibilizados e jogadores que realizam as compras. Assim, um banco de dados é importante para organizar essas informações, permitindo fluidez, consultas, análises e gestão do negócio.


Justificativa das entidades e relacionamentos: 
• Franquia 
A franquia acaba entrando como entidade, pois representa as séries de jogos criados e publicados pela empresa, como por exemplo a série de jogos de The Sims, FIFA ou Need for Speed. Essa entidade é importante porque uma franquia pode possuir vários jogos, permitindo organizar os títulos lançados dentro de uma mesma marca.

• Jogo
A entidade jogo representa cada título desenvolvido ou publicado pela empresa. Ela é essencial, pois concentra informações como nome do jogo, data de lançamento e a franquia na qual pertence, sendo um dos principais elementos.

• Estúdio 
O estúdio retrata os estúdios responsáveis pelo desenvolvimento dos jogos. Essa entidade é necessária para registrar quais equipes ou empresas participaram da criação de cada jogo.

• Funcionário 
A entidade funcionário corresponde aos profissionais envolvidos no desenvolvimento dos jogos, como programadores, designers e produtores. Ela permite registrar quais pessoas participaram dos projetos e em quais jogos trabalharam.

• Jogador
A entidade jogador representa os usuários que compram e utilizam os jogos. Essa entidade permite registrar informações dos clientes e relacionar quais jogos foram adquiridos por cada um dos jogadores.
